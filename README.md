# Node.js 平方根シミュレーター (Square Root Simulator)

Node.js環境で動作する、平方根（ルート）の計算と、その裏側にある計算プロセス（アルゴリズム）を視覚化するためのシミュレーターです。

##  特徴

1. **アルゴリズム視覚化シミュレーター (`simulator.js`)**
   * コンピュータが平方根を割り出す際に使う「ニュートン法（バビロニアの数理）」のプロセスを1ステップずつ出力します。
   * ループごとに予測値がどのように真値に近づいていくか（収束の様子）を観察できます。

2. **対話型シミュレーター (`interactive.js`)**
   * コンソール上でユーザーが入力した任意の数値の平方根を、JavaScriptの標準機能（`Math.sqrt`）を用いて即座に計算します。

---

##  動作環境

* Node.js (v14以上推奨)

---

##  セットアップと実行方法

### 1. プロジェクトの準備
任意のディレクトリを作成し、コードを配置します。

```bash
mkdir square-root-simulator
cd square-root-simulator
上記のディレクトリ内に、該当するスクリプトファイル（simulator.js または interactive.js）を作成・保存してください。2. スクリプトの実行A. ニュートン法の計算プロセスを見る場合Bashnode simulator.js
出力例:Plaintext=========================================
  2 の平方根シミュレーションを開始
=========================================
初期予測値: 1

[ステップ 1]
  現在の予測値: 1.5
  前回の差分  : 5.000000000000000e-1
  -----------------------------------
[ステップ 2]
  現在の予測値: 1.4166666666666667
  ...
B. 自由な数値を入力して計算する場合Bashnode interactive.js
終了したい場合は q を入力してEnterを押します。仕組みについて（ミニ知識）simulator.js で採用しているニュートン法は、以下の漸化式を用いて近似値を求めます。$$x_{n+1} = \frac{1}{2} \left(x_n + \frac{N}{x_n}\right)$$（$N$ は平方根を求めたい数、$x$ は予測値）この計算を繰り返すことで、驚異的な速さで正確な平方根の数値へと収束していきます。シミュレーターのステップごとの「前回の差分」が急激に小さくなっていく様子から、その効率性を実感できます。
